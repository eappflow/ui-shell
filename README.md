# eappflow/ui-shell

This repository conrtains the UI Shell. The UI Shell provides a consistent layout and navigation structure for the application.
It is generic and can be used across different applications within the eAppFlow ecosystem.

It contains application structure for business applications, and also provides a set of reusable components that can be used to build the user interface of the application.

## Packages:
@eappflow/ui-shell - The main package that contains the UI Shell components and layout. Main flow for authentication, navigation, and layout.
@eappflow/ui-shell-components - A package that contains reusable UI components that can be used within the UI Shell. These components include buttons, forms, modals, and other common UI elements.

## Concept:
The libraries deliver configurable and extensible UI components that can be easily integrated into different applications. The UI Shell provides a consistent user experience across applications, while the reusable components allow for flexibility and customization. 
It is designed to easily replace the UI Shell with a different implementation if needed, without affecting the core functionality of the application. This modular approach allows for better maintainability and scalability of the application.
The concept of modules allows them to be independently developed and maintained, enabling better separation of concerns and easier collaboration among development teams. Each module can be developed and tested independently, and then integrated into the main application when ready. This approach promotes code reusability and makes it easier to manage the complexity of large applications.
